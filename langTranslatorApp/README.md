
SIMDB Language Translator App

The SIMDB Language Translator App is a simple Streamlit web application developed with Python that translates English words into selected Nigerian languages.
It was developed as a group project to demonstrate collaboration using Git and GitHub.


How To Run the App
1. Clone the Repository
      "git clone https://github.com/Israel2461/SIMDB-grp-cos101-projects.git"
   
2. Navigate to the project folder
       "cd langTranslatorApp"

3. Install Streamlit (if not installed)
       "pip install streamlit"

4. Run the application
       "streamlit run main.py"
